# YourMentors.io
## Shell Scripting & Python for DevOps

### Course Schedule

| Class No | Topic | Key Topics | Duration |
|---:|---|---|---:|
| 01 | Introduction to Shell Scripting and Fundamentals | History & importance; Shell types (Bash, Zsh); DevOps automation overview; Basic syntax; Running first script; Variables, loops, conditionals; Hello World & I/O exercises | 2 hours |
| 02 | Core Features of Shell Scripting | Control structures; Functions & script organization; File & directory handling; Practical exercises for conditionals, loops, and functions | 2 hours |
| 03 | Advanced Shell Scripting Techniques | Command-line arguments; Error handling & debugging; Regular expressions; Logging; Writing robust scripts with error handling | 2 hours |
| 04 | Shell Scripting for Command-line | Directory stack management; Filesystem operations; Utility functions; Using man pages; Interactive scripting examples | 2 hours |
| 05 | Shell Scripting with System Automation | Text processing & file handling; Advanced file operations; Automating tasks with cron; Real-world automation examples | 2 hours |
| 06 | Python Fundamentals for DevOps | Why Python for DevOps; Installing & environment setup; Syntax, data types, control flow; Basic scripting exercises | 2 hours |
| 07 | Python Scripting for System Administration | File handling; OS & subprocess modules; Error handling; Automating server management tasks with Python | 2 hours |
| 08 | Working with APIs and Web Services in Python | RESTful APIs; `requests` module; JSON/XML handling; Automating API interactions for DevOps | 2 hours |
| 09 | Python for Network Automation | Network automation concepts; SSH with Paramiko/Netmiko; Automating device configuration & monitoring; Intro to Wireshark (bonus) | 2 hours |
| 10 | Combining Shell and Python Scripts | Integrating shell & Python; Hybrid automation; Intro to CI/CD; Automating deployment pipelines with practical pipeline example | 2 hours |
| 11 | DevOps Security Automation with Python | DevSecOps intro; Automating security checks (SAST/DAST); Vulnerability scanning with Python; Securing CI/CD pipelines | 2 hours |
| 12 | Monitoring and Logging with Python | Building monitoring with Prometheus & Python; Log aggregation (ELK); Automating alerts & notifications; Hands-on monitoring exercises | 2 hours |
| 13 | Leveraging Serverless Technologies with Python | Serverless intro; Serverless Framework & multi-cloud; AWS Lambda basics; OpenFaaS self-hosted functions; IaC with AWS CDK (DynamoDB, Lambda, API Gateway) | 2 hours |
| 14 | Exam & Wrap-up | Final exam covering all topics; Best practices; Q&A and feedback | - |
